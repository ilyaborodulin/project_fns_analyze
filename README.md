# Анализ компаний строительной отрасли (ОКВЭД 41.20)

## Описание проекта
В проекте проводится анализ компаний строительной отрасли с основным ОКВЭД 41.20 — строительство жилых и нежилых зданий.

Исходные данные берутся из реестра субъектов МСП. Далее компании фильтруются по заданным условиям, для части выборки через API ФНС получаются идентификаторы организаций (`org_id`) и данные бухгалтерской отчётности (`bfo`).

На основе полученных данных формируется плоская таблица по компаниям и годам, рассчитывается темп роста чистой прибыли и строится график распределения этого показателя.

## Цель проекта
Цель проекта — протестировать пайплайн анализа компаний строительной отрасли: от подготовки реестра и получения финансовых данных до расчёта аналитических показателей и визуализации результатов.

## Что сделано
- отфильтрован реестр компаний:
  - только юридические лица
  - только субъекты МСП
  - только компании с ОКВЭД 41.20
- получены `org_id` для тестовой выборки компаний
- выгружены данные бухгалтерской отчётности по годам
- собрана плоская таблица финансовых показателей
- рассчитан темп роста чистой прибыли
- построена гистограмма распределения роста чистой прибыли

## Логика работы проекта

## 1. Подготовка реестра

## Скрипт prepare_registry.py:
	•	читает исходный Excel-файл реестра
	•	оставляет только:
	•	юридические лица
	•	субъекты МСП
	•	компании с основным ОКВЭД 41.20
	•	сохраняет результат в data/registry_filtered.csv

## 2. Получение org_id

## Скрипт get_org_ids_sample.py:
	•	берёт тестовую выборку компаний из registry_filtered.csv
	•	по ИНН делает запросы к сервису ФНС
	•	получает org_id организаций
	•	сохраняет результат в data/sample_with_org_ids.csv

## 3. Получение бухгалтерской отчётности

## Скрипт extract_bfo_sample.py:
	•	по org_id обращается к endpoint с бухгалтерской отчётностью
	•	извлекает данные по годам
	•	формирует плоскую таблицу финансовых показателей
	•	сохраняет её в data/flat_sample_30_companies.csv

## 4. Анализ роста чистой прибыли

## Скрипт analyze_growth_sample.py:
	•	рассчитывает темп роста чистой прибыли по годам
	•	строит гистограмму распределения роста
	•	сохраняет:
	•	data/flat_sample_30_companies_with_growth.csv
	•	outputs/net_profit_growth_hist_sample.png

## Основные результаты
	•	исходная фильтрация по реестру дала 142930 компаний
	•	в тестовой выборке из 100 компаний удалось получить 98 org_id
	•	для анализа bfo была взята выборка из 30 компаний
	•	по ним получено 118 наблюдений по годам
	•	темп роста чистой прибыли рассчитан для 70 наблюдений
	•	после отсечения выбросов для графика осталось 58 наблюдений
	•	минимальное значение роста чистой прибыли: -207.15%
	•	максимальное значение роста чистой прибыли: 489.30%

## Аналитический вывод

На тестовой выборке из 30 компаний удалось собрать финансовые данные по годам и рассчитать темп роста чистой прибыли для части наблюдений.

Распределение роста чистой прибыли оказалось неоднородным: в выборке присутствуют как положительные, так и отрицательные значения. Это показывает, что внутри исследуемой выборки нет единой общей тенденции: часть компаний демонстрирует рост чистой прибыли, а часть — снижение.

Следовательно, финансовое состояние компаний в рассматриваемом сегменте строительной отрасли является неоднородным.

## Ограничения проекта
	•	анализ выполнен на тестовой выборке, а не на всей совокупности компаний
	•	не для всех наблюдений доступны все финансовые показатели
	•	расчёт EBITDA пока не реализован из-за неполноты нужных полей в части отчётностей
	•	выводы носят предварительный характер, так как основаны на ограниченной выборке
